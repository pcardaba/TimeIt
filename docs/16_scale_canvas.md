# How to scale waveform canvas

Scaling and scrolling the waveform canvas is done entirely with the mouse wheel — no toolbar buttons or menu entries are needed.

## Scaling (x-axis only)

Scaling is only supported along the **x-axis** (time axis). It is not possible to scale the canvas vertically.

| Action | Effect |
|--------|--------|
| **Shift + Mouse Wheel up** | Zoom in (expand the time axis) |
| **Shift + Mouse Wheel down** | Zoom out (compress the time axis) |

## Scrolling (y-axis only)

The canvas can be scrolled vertically to reveal signals that are outside the visible area.

| Action | Effect |
|--------|--------|
| **Mouse Wheel up** | Scroll canvas up |
| **Mouse Wheel down** | Scroll canvas down |
